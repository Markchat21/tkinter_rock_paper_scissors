# tkinter_rock_paper_scissors

Implementación simple de piedra, papel o tijeras en tkinter(python).
Proyecto creado para indagar superficialmente en el uso de interfaces graficas de escritorio(GUI)

# ¿Cómo puedo ejecutarlo?

1. Clona este repositorio en tu máquina local.
2. Ejecuta main.py en tu entorno de desarrollo de python
